# SDES
Implementation of Simplified Data Encryption Standard


## From Terminal
No logging - > Log/SDES.exe
logging    - > Log/SDES.exe

## Input Form
Encryptrion: SDES [E] [input secret Key] [Plain Text]
Decryption : SDES [D] [input secret Key] [Cypher Text]
